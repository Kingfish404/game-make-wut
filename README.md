# Make WUT

合成武汉理工小游戏，Let's make WUT great again (Doge

使用[cocos](https://www.cocos.com/),在gitee[开源仓库](https://gitee.com/cocoscreatorfree/cocos-big-watermelon)的基础上二次开发

## 修改部分

* 添加了计分功能
* 修改界面
* 重写了操控逻辑和动作(cocos的Action系统即将弃用，要升级为tween缓动系统)
